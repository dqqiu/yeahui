yeahui-message信息框组件(弹出层)
此模块名称：message，源代码托管：https://www.github.com/dqqiu/yeahui
使用说明

message信息框弹层组件使用说明如下：

参数名 参数意义 默认值 说明
id 弹层唯一标识号 "",空字符串 标识弹层唯一，未指定时从0开始递增，若中途有指定过id，将跳过其原来应有的标识号，如：在第一个未指定id，第二个弹层id指定为xxx，第三个弹层不指定id，则id标识号分别为0，xxx，2，将跳过标识号1
minWidth 可调整的窗口最小宽度(启用resize功能时有效) 各类型弹层默认值不相同，此处不一一说明 可调整的窗口最小宽度，只在resize功能时有效
minHeight 可调整的窗口最小高度(启用resize功能时有效) 各类型弹层默认值不相同，此处不一一说明 可调整的窗口最小高度，只在resize功能时有效
width 弹窗宽度 各类型弹层默认值不相同，此处不一一说明 弹窗宽度
height 弹窗高度 各类型弹层默认值不相同，此处不一一说明 弹框高度
draggable 是否可拖拽(true/false) false true：可进行拖拽，false：不可拖拽
resizeable 是否可调整窗口尺寸(true/false) false true：可进行窗口尺寸调整，false：不可进行窗口尺寸调整
maxButton 是否启用最大化按钮(true/false) false true：启用最大化按钮，false：不启用最大化按钮，启用后可进行窗口最大化操作，窗口最大化后按钮作用改变为恢复原窗口大小
modal 是否模态化显示(true/false) true 配置弹窗是否模态化显示，模态化显示时则弹窗背后出现遮罩层
modalClose 是否支持点击模态遮罩层关闭弹窗(true/false) false 是否支持点击模态遮罩层关闭弹窗，true：点击遮罩层关闭弹窗，只在启用模态化显示时生效
showTime 弹窗显示时间，单位：毫秒 0 弹窗显示时间，单位为毫秒，设置了该值后窗口在指定的时间后将被关闭
buttons 弹窗按钮 各类型不一致 弹窗按钮，可配置多个(alert弹层只支持一个)，数组类型，子对象属性---text:按钮文字，close:点击按钮后是否关闭,class:按钮样式配置,callback:点击按钮后的回调函数
alert弹层演示

下列为各种不同参数配置的alert弹层，点击试试看吧

				  yeahui.require(["jquery", "message", "ccode"], function(exports) {
				    var message = yeahui.message(),
					    $ = yeahui.jquery,
					    ccode = yeahui.ccode();
							
				    $("#default-alert").click(function(e) {
				      message.alert("默认alert弹层");
				    });

				    $("#modal-close-alert").click(function(e) {
				      message.alert("点击遮罩可关闭的alert弹层", {modalClose : true});
				    });

				    $("#merge-alert").click(function(e) {
				      message.alert("可最大化可调整尺寸可拖拽", {
				        maxButton : true,
				        resizeable : true,
				        draggable : true
				      });
				    });
				  });
